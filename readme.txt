== Description ==

Allow users to use "Image" column type with the plugin "acf-image-crop-add-on". You will see as content in the column the cropped image you've created.

== Installation ==

1. Upload the file to plugins folder
2. Activate the plugin in the Wordpress admin panel
3. Configure the Codepress Admin Columns plugin to show Image field type with a cropped image.
4. See the result
